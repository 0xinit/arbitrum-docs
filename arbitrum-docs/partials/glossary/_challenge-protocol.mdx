<p>
The protocol by which assertions are submitted, disputed, and ultimately confirmed. The Challenge Protocol guarantees that only valid <a href="/intro/glossary#assertion">Assertion</a> will be confirmed provided that there is at least one honest <a href="/intro/glossary#active-validator">Active Validator</a>.
</p>
